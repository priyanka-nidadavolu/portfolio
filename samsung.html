Phoneme Data Creation

Enhancing Speech Recognition Accuracy with Multilingual Phonetic Dataset

  • Duration: Jun 2021 - Mar 2022
  • Institution:Samsung Prism
  • Tech Stack: Python NumPy and Pandas Soundex SpeechRecognition Library

Problem Statement

Speech recognition systems often struggle with accurately interpreting names due to the diversity in linguistic phonetics, especially in multilingual settings. This limitation can lead to incorrect recognition or transcription, impacting user experience and accessibility. The challenge was to create a robust solution that improves the accuracy of speech recognition systems, focusing on names from diverse languages and dialects.


Objective

To develop a multilingual dataset with phonetic representations for enhancing the accuracy of speech recognition systems by mitigating linguistic and phonetic ambiguities.

About the Project

Dataset Development:
Compiled a dataset of 10,000+ names across multiple languages, ensuring broad linguistic coverage.
Annotated each name with its phonetic representations using the International Phonetic Alphabet (IPA).

Phonetic Matching Algorithms:
Implemented phonetic matching techniques to align audio inputs with textual representations.
Developed validation scripts to test and refine phonetic consistency.

Data Processing and Automation:
Used Python libraries such as Pandas for data manipulation and NumPy for numerical operations.
Automated processes for data cleaning, validation, and dataset augmentation.

Programming Languages:
Programming Language:
Python: Core language for scripting, data processing, and phonetic algorithm implementation.

Libraries and Tools:
SpeechRecognition Library: For analyzing speech patterns and testing the dataset integration.
NumPy and Pandas: For numerical computation and dataset management.
Phonetic Algorithms:
Leveraged Soundex and Metaphone algorithms to enhance phonetic similarity matching.
Integrated Phoneme Recognition Techniques for precise alignment.

Improved Speech Recognition Accuracy:
Achieved a 10% improvement in accuracy for speech recognition systems when tested with the dataset.
Enhanced the system's ability to interpret names across diverse languages and dialects.

Contribution to Accessibility:
Enhanced user experience by reducing recognition errors in applications relying on speech-to-text functionality.

Scalable Framework:
Provided a framework for augmenting speech recognition systems with more diverse and extensive datasets in the future.

Innovation in Multilingual Systems:
Positioned the project as a foundational contribution to improving accessibility in multilingual environments, impacting a broad range of applications from virtual assistants to automated customer service systems.

This project not only addressed a critical gap in speech recognition systems but also laid the groundwork for future enhancements in multilingual and culturally diverse voice technologies.
